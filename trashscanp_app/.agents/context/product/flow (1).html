⚡ Error & Edge Case Paths
  • Tidak ada koneksi → Snackbar error, loop tetap jalan (akan retry interval berikutnya)
  • API timeout (5s) → Skip frame, log error, lanjut loop
  • 503 cold start → Pause loop, banner "Model dimuat...", resume setelah estimated_time
  • Request pending → Skip frame (flag _isRequesting)
  • Kamera izin ditolak → PermissionDeniedScreen khusus
  • App background → Loop auto-pause via WidgetsBindingObserver

05 — API Contract
POST https://api-inference.huggingface.co/models/{MODEL_NAME}
// Headers
Authorization: Bearer $HF_API_TOKEN
Content-Type: application/octet-stream
// Body — raw compressed frame bytes (<300KB)
[binary image data dari camera capture]
// Response 200 OK
[
  { "label": "plastic", "score": 0.92 },
  { "label": "organic", "score": 0.05 }
]
// → ambil index [0] (score tertinggi) sebagai topLabel
// Response 503 — model cold start
{ "error": "Model is loading", "estimated_time": 20.0 }
// → pause loop, status = coldStart, resume setelah estimated_time
// Timeout: 5 detik per request
// On timeout/error: skip frame, _isRequesting = false, lanjut loop
// Tidak ada retry langsung — tunggu interval berikutnya (2s)
