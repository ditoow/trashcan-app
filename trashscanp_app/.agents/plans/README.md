# plans/

Implementation plans — **how** to build it, step by step.

**Naming:** `YYYY-MM-DD-<feature-slug>.md` (slug sama dengan spec terkait)
**Template:** copy `000-TEMPLATE.md`

Update checklist real-time saat task selesai: `- [ ]` → `- [x]`
Setelah semua done: isi `## Walkthrough`, pindahkan ide out-of-scope ke backlog di `logs/fixes/LOG.md`
