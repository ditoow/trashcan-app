# skills/

Project-specific Agent Skills. Each skill follows the format:

```
skills/
└── <skill-name>/
    └── SKILL.md    # frontmatter (name, description, triggers) + instructions
```

Pre-installed skills (generated by init-agents):
- `feature-workflow/` — mandatory flow for new features
- `error-check/` — verify build/lint/test after coding

Add project-specific skills here as reusable workflows emerge.
